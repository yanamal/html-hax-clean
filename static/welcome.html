Welcome!

HTML Hax is a website which teaches you HTML through a series of "hacking" style puzzles.
Each puzzle is a web page. To solve it, you must:
  1. Read the source HTML (and sometimes other code)
  2. Find the password (or find another way to get through the puzzle)
  3. Submit the password using the form (or sometimes through other hacks!)
Not sure how you'd do any of those things? don't worry, the first puzzle shows you!
